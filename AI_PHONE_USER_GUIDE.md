# AI电话工坊使用指南

## 快速开始

### 第一步：进入AI电话工坊

1. 打开应用首页
2. 点击导航栏的"情景训练"
3. 选择"AI电话工坊"标签
4. 浏览可用的训练场景

### 第二步：选择训练场景

我们提供多个真实商务场景：

- 🇲🇾 **马来西亚客户电话咨询**（初级，15分钟）
- 🇵🇭 **菲律宾供应商电话谈判**（中级，25分钟）
- 🇹🇭 **泰国合作伙伴紧急沟通**（高级，20分钟）
- 🇸🇬 **新加坡客户售后服务**（中级，20分钟）

点击任意场景的"开始训练"按钮。

### 第三步：开始通话

1. 点击"开始通话"按钮
2. 阅读AI的欢迎语
3. 选择输入模式：
   - **语音模式**：使用麦克风说话（推荐）
   - **文字模式**：键盘输入英语

## 语音模式使用

### 准备工作

✅ **浏览器要求**：Chrome 90+ 或 Edge 90+
✅ **设备要求**：带麦克风的电脑或手机
✅ **环境要求**：相对安静的环境
✅ **权限要求**：允许浏览器访问麦克风

### 录音步骤

1. **点击"点击录音"按钮**
   - 浏览器会请求麦克风权限
   - 首次使用需要点击"允许"

2. **开始说话**
   - 用英语说出你的回复
   - 至少说2秒以上
   - 最长可录60秒
   - 屏幕会显示录音时长

3. **停止录音**
   - 点击"停止录音"按钮
   - 或等待60秒自动停止

4. **等待处理**
   - 系统会显示"正在处理音频，请稍候..."
   - 通常需要5-10秒
   - 处理完成后会显示识别结果

5. **查看AI回复**
   - AI会用英语回复你
   - 同时播放语音
   - 你的分数会增加

### 语音模式技巧

💡 **说话技巧**
- 清晰发音，不要含糊
- 语速适中，不要太快
- 靠近麦克风，保持稳定距离
- 避免背景噪音

💡 **内容建议**
- 使用完整的句子
- 保持商务礼貌用语
- 注意文化差异
- 参考场景描述

💡 **常见问题**
- 如果识别不准确，重新录制
- 如果环境嘈杂，切换到文字模式
- 如果网络不好，等待自动重试

## 文字模式使用

### 何时使用文字模式

- 🔇 环境太吵，无法录音
- 🎤 没有麦克风设备
- 🌐 网络不稳定
- 📝 想要更精确的表达
- 🧪 测试功能是否正常

### 使用步骤

1. **切换模式**
   - 点击"切换到文字输入"按钮
   - 录音按钮会变成输入框

2. **输入文字**
   - 在输入框中输入英语
   - 例如：`Hello, how can I help you?`
   - 支持多行输入

3. **发送消息**
   - 点击发送按钮（纸飞机图标）
   - 或按Enter键（桌面端）

4. **查看回复**
   - AI会立即回复
   - 同样会播放语音
   - 分数会增加

### 文字模式优势

✅ 不受环境噪音影响
✅ 可以仔细组织语言
✅ 响应速度更快（2-5秒）
✅ 适合学习和练习

## 对话技巧

### 商务英语要点

1. **礼貌用语**
   - 开场：`Good morning/afternoon, this is...`
   - 请求：`Could you please...?`
   - 感谢：`Thank you for your time.`
   - 结束：`Looking forward to hearing from you.`

2. **清晰表达**
   - 使用简单句式
   - 避免复杂从句
   - 重要信息重复确认
   - 使用专业术语

3. **文化适应**
   - 泰国：尊重长辈，避免直接拒绝
   - 新加坡：注重效率，直接沟通
   - 马来西亚：建立关系，耐心倾听
   - 菲律宾：友好热情，重视人际关系

### 对话示例

#### 场景：马来西亚客户咨询

**你**: I'm interested in your textile products, specifically cotton fabrics.

## 常见问题

### Q1: 麦克风无法使用怎么办？

**A**: 检查以下几点：
1. 浏览器是否允许麦克风权限
2. 麦克风设备是否正常连接
3. 其他应用是否占用麦克风
4. 尝试刷新页面重新授权
5. 如果仍然不行，使用文字模式

### Q2: 语音识别不准确怎么办？

**A**: 改进识别准确度：
1. 在安静环境中录音
2. 靠近麦克风说话
3. 清晰发音，语速适中
4. 使用标准英语发音
5. 避免方言和俚语

### Q3: 网络连接失败怎么办？

**A**: 系统会自动重试：
1. 第一次失败：等待1秒后重试
2. 第二次失败：等待2秒后重试
3. 第三次失败：显示错误提示
4. 检查网络连接后重试
5. 或切换到文字模式

### Q4: 录音时间太短怎么办？

**A**: 确保录音至少2秒：
1. 说完整的句子
2. 不要只说单个单词
3. 观察录音时长提示
4. 如果不小心停止，重新录制

### Q5: AI回复不合理怎么办？

**A**: 这是正常的学习过程：
1. AI基于上下文生成回复
2. 可能需要多轮对话才能理解
3. 尝试更清晰地表达你的意图
4. 参考场景描述调整对话方向

### Q6: 如何提高训练效果？

**A**: 最佳实践：
1. 每天练习15-30分钟
2. 尝试不同难度的场景
3. 录音模式和文字模式结合使用
4. 注意AI的回复，学习表达方式
5. 记录不熟悉的词汇和表达
