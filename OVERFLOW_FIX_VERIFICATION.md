## 常见问题

### Q1: 手机端Tab看不全怎么办？

**A**: Tab导航支持横向滑动，用手指左右滑动即可查看所有Tab。

### Q2: 文字还是被裁切怎么办？

**A**: 
1. 强制刷新浏览器（Ctrl+F5或Cmd+Shift+R）
2. 清除浏览器缓存
3. 检查是否是最新版本的代码

### Q3: Dialog还是超出屏幕怎么办？

**A**: 
1. 检查浏览器缩放比例（应为100%）
2. 检查设备像素比
3. 尝试在其他浏览器中打开

### Q4: 桌面端显示变小了怎么办？

**A**: 不会。桌面端使用`md:max-w-4xl`保持原有宽度（896px）。

## 技术细节

### 响应式断点

```css
/* 手机端（默认） */
max-w-[95vw]    /* Dialog宽度 */
text-xs         /* 字体大小 */
p-2             /* 内边距 */
h-3 w-3         /* 图标大小 */

/* 桌面端（≥ 768px） */
md:max-w-4xl    /* Dialog宽度 */
md:text-sm      /* 字体大小 */
md:p-4          /* 内边距 */
md:h-4 md:w-4   /* 图标大小 */
```

### 关键CSS类

```css
/* 防止溢出 */
overflow-hidden
overflow-x-auto
max-w-[95vw]

/* 文字换行 */
break-words
line-clamp-2
line-clamp-3

/* Flex布局 */
flex flex-col
flex-1
flex-shrink-0

/* 响应式 */
text-xs md:text-sm
p-2 md:p-4
h-3 w-3 md:h-4 md:w-4
```
