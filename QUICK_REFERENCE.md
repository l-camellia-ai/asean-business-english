## ⚡ 快速测试（5分钟）

### 步骤1：测试文字模式
```
1. 打开应用 → 情景训练 → AI电话工坊
2. 选择任意场景 → 开始训练 → 开始通话
3. 点击"切换到文字输入"
4. 输入：Hello, how are you?
5. 点击发送
6. ✅ 看到AI回复 = 成功
```

### 步骤2：测试语音模式
```
1. 点击"切换到语音识别"
2. 允许麦克风权限
3. 点击"点击录音"
4. 说英语至少2秒
5. 点击"停止录音"
6. ✅ 看到识别结果和AI回复 = 成功
```

## 🔍 常见问题速查

| 问题 | 解决方案 |
|------|----------|
| 文字模式失败 | 检查API密钥配置 |
| 麦克风无法访问 | 检查浏览器权限 |
| 识别不准确 | 安静环境、清晰发音 |
| 录音时间太短 | 至少说2秒以上 |
| 网络连接失败 | 检查网络，系统会自动重试 |
