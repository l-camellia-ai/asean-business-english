## 二、用户流程

### 流程1：新用户注册
```
1. 访问 /login
2. 点击"注册"标签
3. 选择注册方式（手机号/邮箱）
4. 填写信息（手机号/邮箱 + 用户名 + 密码）
5. 点击"注册"
6. 自动跳转到 /onboarding
7. 完成3步引导流程
8. 跳转到首页，开始学习
```

### 流程2：老用户登录
```
1. 访问 /login
2. 默认在"登录"标签
3. 选择登录方式（用户名/手机号/邮箱）
4. 输入凭证 + 密码
5. 点击"登录"
6. 跳转到首页
```

### 流程3：游客体验
```
1. 访问 /login
2. 点击"游客体验（免注册试用）"按钮
3. 直接跳转到首页
4. 可以体验1-2个核心场景
5. 适时弹出注册引导（待实现）
```
