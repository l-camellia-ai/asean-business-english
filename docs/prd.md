# 需求文档

## 一、应用概述

### 1.1 应用名称
东盟商务英语训练

### 1.2 应用描述
面向东盟市场的商务英语学习与训练平台H5应用,提供情景实战、商务知识库、学习社区等功能模块,支持个人学习与企业培训场景。新增启动页(登录/注册/游客体验入口)、新用户引导流程,实现个性化入门体验。核心功能新增AI数字人交互,支持多国数字人形象切换与口型同步。

---

## 二、页面布局设计

### 2.1 顶部导航区(4个入口)

**1. 情景实战**
- 定位:平台核心功能,放在首位
- 子功能:AI情景对话工坊、AI情景电话工坊、标准化情景模块训练
- 适用用户:全体用户
- 亮点:突出多口音模拟、高仿真谈判等核心卖点

**2. 商务知识库**
- 定位:知识输入与系统学习
- 子功能:国别商务文化指南、行业术语与实务库、动态政策与市场洞察
- 适用用户:全体用户
- 亮点:短视频、信息图、互动案例等多媒体呈现

**3. 学习社区**
- 定位:社交激励与成长展示
- 子功能:个人成长展示墙、多维激励排行榜、专家问答
- 适用用户:全体用户
- 亮点:一键分享成就卡片、专家直播预告

**4. 企业服务(仅企业用户可见)**
- 定位:B端专属入口
- 子功能:团队管理后台、能力诊断报告、定制化方案管理
- 适用用户:企业管理员
- 亮点:团队能力地图、培训效果ROI报告

### 2.2 首页核心推荐区(2个入口)

**5. 我的学习地图**
- 定位:个人专属学习进度与规划
- 呈现形式:东盟市场开拓地图可视化
- 功能:当前关卡、待解锁区域、经验值进度、勋章收集
- 亮点:六维能力雷达图实时更新

**6. 每日挑战**
- 定位:游戏化激励入口,提升日活
- 呈现形式:每日任务卡片(如完成1次越南口音电话谈判)
- 功能:签到任务、限时挑战、连胜奖励
- 亮点:完成即可获得双倍经验值

### 2.3 用户中心区(2个入口)

**7. AI实训增强包**
- 定位:增值服务转化入口
- 呈现形式:会员权益卡片
- 功能:解锁无限次AI情景工坊评分、深度评测报告
- 亮点:当前使用次数/额度显示、订阅引导

**8. 个人中心**
- 定位:账户管理与设置
- 功能:个人信息、学习设置、订阅管理、企业绑定
- 呈现形式:头像+下拉菜单或独立页面

## 三、功能模块说明

### 3.1 启动页

**页面定位**
- 应用首次打开或未登录状态下的入口页,提供登录、注册与游客体验三条路径

**页面结构**
- 顶部:应用Logo与品牌名称
- 中部:三个主操作按钮
  - 「登录」:跳转至登录页
  - 「注册」:跳转至注册页
  - 「先体验,后注册」:以游客身份进入游客体验模式
- 底部:用户协议与隐私政策说明文案

**交互逻辑**
- 已登录用户直接跳过启动页,进入首页
- 游客点击「先体验,后注册」后进入游客体验模式,可免费试用1-2个核心对话场景
- 游客体验过程中,在适当时机(如完成一个场景后或触发付费功能时)弹出注册引导弹窗,引导用户完成注册

---

### 3.2 登录页

**页面定位**
- 已有账号用户的登录入口

**页面结构**
- 顶部:页面标题「登录」
- 中部:登录表单区域
- 底部:跳转注册链接、忘记密码链接

**登录方式**
- 手机号 + 密码登录(支持国内及东盟国家手机号,含国际区号选择)
- 邮箱 + 密码登录
- 微信授权快捷登录

**交互逻辑**
- 手机号登录时,提供国际区号下拉选择(默认+86,支持东盟主要国家区号)
- 登录成功后跳转至首页
- 登录失败时在对应字段下方展示错误提示文案
- 点击「忘记密码」跳转至密码重置流程

**业务规则**
- 微信授权登录成功后,若该微信未绑定账号,则引导用户完成手机号或邮箱绑定后再进入首页

**异常与边界情况**

| 异常场景 | 处理方式 |
|---|---|
| 账号不存在 | 提示「账号不存在,请先注册」,并提供跳转注册链接 |
| 密码错误 | 提示「密码错误,请重新输入」 |
| 网络异常 | 提示「网络异常,请稍后重试」 |
| 微信授权失败 | 提示「微信授权失败,请重试或选择其他登录方式」 |

---

### 3.3 注册页

**页面定位**
- 新用户创建账号的入口,支持多种注册方式

**页面结构**
- 顶部:页面标题「创建账号」
- 中部:注册方式选项卡 + 注册表单区域
- 底部:已有账号登录跳转链接

**注册方式**

方式一:手机号注册
- 国际区号选择(默认+86,支持东盟主要国家区号)
- 手机号码(必填)
- 验证码(必填,发送至手机号,有效期5分钟)
- 登录密码(必填,8-20位,需包含字母与数字)
- 确认密码(必填,与登录密码一致)
- 用户类型选择(必选:个人用户 / 企业用户)
  - 选择「企业用户」时,额外展示:企业名称(必填)、企业邀请码(选填)
- 同意用户协议与隐私政策勾选框(必选)

方式二:邮箱注册
- 邮箱地址(必填)
- 验证码(必填,发送至邮箱,有效期5分钟)
- 登录密码(必填,8-20位,需包含字母与数字)
- 确认密码(必填,与登录密码一致)
- 用户类型选择(必选:个人用户 / 企业用户)
  - 选择「企业用户」时,额外展示:企业名称(必填)、企业邀请码(选填)
- 同意用户协议与隐私政策勾选框(必选)

方式三:微信快捷注册
- 点击「微信快捷注册」按钮,发起微信授权
- 授权成功后,引导用户补充绑定手机号或邮箱(必填,用于账号唯一标识)
- 用户类型选择(必选:个人用户 / 企业用户)
  - 选择「企业用户」时,额外展示:企业名称(必填)、企业邀请码(选填)
- 同意用户协议与隐私政策勾选框(必选)

**交互逻辑**
- 点击「获取验证码」按钮后,按钮进入60秒倒计时冷却状态,冷却结束后可重新发送
- 表单提交前进行前端校验:手机号/邮箱格式、密码强度、两次密码一致性、必填项是否填写
- 注册成功后自动跳转至新用户引导流程
- 注册失败时,在对应字段下方展示错误提示文案

**业务规则**
- 同一手机号或邮箱仅允许注册一个账号
- 企业用户填写有效邀请码后,注册完成自动绑定对应企业
- 用户协议与隐私政策未勾选时,禁止提交注册

**异常与边界情况**

| 异常场景 | 处理方式 |
|---|---|
| 手机号/邮箱已注册 | 提示「该账号已注册,请直接登录」,并提供跳转登录链接 |
| 验证码错误或过期 | 提示「验证码错误或已过期,请重新获取」 |
| 两次密码不一致 | 提示「两次输入的密码不一致,请重新输入」 |
| 网络异常 | 提示「网络异常,请稍后重试」 |
| 企业邀请码无效 | 提示「邀请码无效,请确认后重新输入」,不阻断注册流程,以个人用户身份完成注册 |
| 微信授权失败 | 提示「微信授权失败,请重试或选择其他注册方式」 |

### 3.4 新用户引导流程

**页面定位**
- 新用户完成注册后,首次进入应用时触发的引导问卷,用于收集用户基本信息,生成个性化初始课程推荐

**触发条件**
- 仅在用户完成注册后首次进入应用时触发,后续登录不再展示
- 游客完成注册后同样触发此流程

**页面结构**
- 顶部:进度指示器(如第1步/共3步)
- 中部:当前问题与选项
- 底部:「下一步」/「完成」按钮,以及「跳过」文字链接

**引导问题(共3题)**

第1题:您的身份是?(单选,必选)
- 企业员工
- 学生
- 创业者
- 其他(选填文本输入)

第2题:您目前的英语水平?(单选,必选)
- 初级(日常沟通有困难)
- 中级(能进行基本商务沟通)
- 高级(能流利进行商务谈判)

第3题:您的主要学习目标是?(单选,必选)
- 商务谈判
- 邮件沟通
- 文化了解
- 其他(选填文本输入)

**交互逻辑**
- 每题回答后点击「下一步」进入下一题,最后一题点击「完成」提交
- 用户可点击「跳过」直接进入首页,跳过后系统使用默认推荐方案
- 完成全部问题后,系统根据回答结果在首页「我的学习地图」中推荐初始课程与关卡

**业务规则**
- 引导问题的回答结果存储至用户档案,用于个性化推荐
- 跳过引导的用户,系统以「中级 + 商务谈判」作为默认推荐基准
- 引导流程完成或跳过后,均跳转至首页

---

### 3.5 游客体验模式

**页面定位**
- 未注册用户可免费体验1-2个核心对话场景,降低获客门槛

**可体验内容**
- 开放「情景实战」模块中的1-2个固定核心对话场景供游客免费试用
- 其余功能模块(商务知识库、学习社区、企业服务、我的学习地图、每日挑战、AI实训增强包、个人中心)在游客模式下均不可访问,点击时弹出注册引导弹窗

**注册引导弹窗触发时机**
- 游客完成一个体验场景后
- 游客尝试访问非开放功能时
- 游客尝试查看评分或详细报告时

**注册引导弹窗内容**
- 标题:「注册即可解锁全部功能」
- 简要说明注册后可获得的核心权益(如:完整课程、学习地图、每日挑战等)
- 操作按钮:「立即注册」(跳转注册页)、「暂不注册」(关闭弹窗,继续游客模式)

**业务规则**
- 游客体验数据(如对话记录)不做持久化存储,注册后重新开始
- 游客模式下不展示个人中心、学习进度等需登录态的功能入口

### 3.6 情景实战模块

**模块定位**
- 平台核心功能,提供AI驱动的商务英语情景训练

**子功能**
- AI情景对话工坊:提供多口音模拟的高仿真谈判场景
- AI情景电话工坊:训练电话沟通能力
- 标准化情景模块训练:覆盖各类商务场景训练

**AI数字人交互功能(新增)**

页面结构:
- 左侧或上方:AI数字人形象展示区域
- 右侧或下方:对话内容展示区域与用户输入区域
- 顶部:数字人国家切换入口

数字人形象:
- 默认展示统一的数字人形象
- 支持切换5个国家的数字人形象:越南、新加坡、马来西亚、印度尼西亚、菲律宾
- 越南数字人形象使用用户提供的图片 676bfcf6f423099bb01586a4b2b6cd9.jpg
- 其余4个国家的数字人形象使用统一的默认形象

数字人交互:
- 用户选择对话场景后,AI数字人自动开始对话
- 数字人回复时,嘴部需呈现与语音同步的口型动画
- 口型动画需与语音内容实时匹配,确保视觉与听觉一致性

数字人切换:
- 用户可通过顶部切换入口选择不同国家的数字人
- 切换后,数字人形象与口音同步更新
- 切换不影响当前对话进度,对话内容保持连续

业务规则:
- 数字人口型动画需与AI语音输出实时同步
- 数字人形象切换后,对话口音自动匹配所选国家
- 游客模式下可体验数字人交互功能,但仅限开放的1-2个场景

## 四、业务规则与逻辑

### 4.1 登录态与页面访问规则

| 用户状态 | 可访问页面 | 限制说明 |
|---|---|---|
| 未登录(游客) | 启动页、登录页、注册页、游客体验(1-2个固定场景) | 其余功能触发注册引导弹窗 |
| 已登录(个人用户) | 除企业服务外的全部功能 | 企业服务入口不可见 |
| 已登录(企业用户) | 全部功能 | 企业服务入口可见 |

### 4.2 新用户引导触发规则
- 仅首次注册完成后触发一次,后续登录不再触发
- 游客完成注册后同样触发
- 跳过引导的用户使用默认推荐基准(中级 + 商务谈判)

### 4.3 微信授权账号绑定规则
- 微信授权登录/注册成功后,若未绑定手机号或邮箱,必须完成绑定后方可进入首页
- 同一微信账号仅可绑定一个平台账号

### 4.4 AI数字人交互规则
- 数字人口型动画必须与AI语音输出实时同步
- 数字人形象切换后,对话口音自动匹配所选国家
- 数字人切换不影响当前对话进度,对话内容保持连续
- 游客模式下可体验数字人交互,但仅限开放的1-2个场景

## 五、异常与边界情况

| 异常场景 | 处理方式 |
|---|---|
| 游客尝试访问受限功能 | 弹出注册引导弹窗,提供「立即注册」与「暂不注册」选项 |
| 游客体验场景完成后 | 自动弹出注册引导弹窗 |
| 新用户引导跳过 | 使用默认推荐基准,直接进入首页 |
| 微信授权后未完成手机号/邮箱绑定 | 不允许进入首页,停留在绑定引导页 |
| 数字人口型动画加载失败 | 显示静态数字人形象,语音正常播放 |
| 数字人形象切换失败 | 保持当前数字人形象,提示「切换失败,请重试」 |
| 网络异常(全局) | 提示「网络异常,请稍后重试」 |

## 六、验收标准

1. 启动页正确展示「登录」「注册」「先体验,后注册」三个入口
2. 已登录用户打开应用直接进入首页,不展示启动页
3. 登录页支持手机号+密码、邮箱+密码、微信授权三种登录方式,登录成功后跳转首页
4. 注册页支持手机号、邮箱、微信快捷注册三种方式,注册成功后进入新用户引导流程
5. 手机号注册与登录均支持国际区号选择,覆盖国内及东盟主要国家
6. 微信授权登录/注册后,若未绑定手机号或邮箱,强制引导完成绑定
7. 新用户引导流程展示3道问题,支持逐题作答与跳过,完成或跳过后进入首页
8. 新用户引导仅在首次注册后触发一次,后续登录不再展示
9. 游客模式下可免费体验1-2个固定核心对话场景,其余功能触发注册引导弹窗
10. 注册引导弹窗在游客完成体验场景后或触发受限功能时正确弹出
